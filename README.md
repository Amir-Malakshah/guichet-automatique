# Guichet Automatique

A desktop banking application built with:

- WPF (.NET)
- ASP.NET Core Web API (.NET 8)
- Azure SQL Database
- Render deployment
- SQL Server

## Architecture

WPF Desktop App → ASP.NET Core API → Azure SQL Database

---

## Features

- Client login
- Create client
- Account blocking
- REST API
- Swagger documentation
